How to use this application with keyboard

Application is accessible for keyboard users. All content should be accessible for screen readers and other assistive technologies.

Editor shortcuts

For editor which was created for write posts, application defined these shortcuts:
  • Alt+Shift+T for focus editor toolbar
  • Alt+Shift+E for focus editor
  • Ctrl+Z for Undo last change
  • Ctrl+Y for Redo last change
  • Ctrl+B for bold text
  • Ctrl+I for italic text
  • Ctrl+U for underline text
  • Ctrl+Shift+L for format align left
  • Ctrl+Shift+R for format align right
  • Ctrl+Shift+E for format align center
  • Ctrl+Shift+J for format align justify
All shortcuts are also in button labels on editor toolbar.
